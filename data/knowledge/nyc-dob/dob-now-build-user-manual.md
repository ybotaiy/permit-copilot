# DOB NOW: Build Job Filing Industry User Manual

This user manual is a dynamic document that is continually edited and updated. Please check the New York City Department of Buildings website to download the most current user manual.

## Introduction

DOB NOW: Build has been designed to allow Owners, Registered Architects, Professional Engineers, Licensees, Filing Representatives, Progress Inspectors, and Special Inspectors to interact with the Department of Buildings in a more efficient manner.

After registering for an account, users can submit applications, make payments, schedule appointments, check the status of an application, pull permits, and make renewals.

## Manual Overview

The manual provides step-by-step instructions for industry users to create and track job filing applications through DOB NOW: Build. It is organized into sections that correspond to the forms users complete when they submit applications, and it includes screenshots to guide users through the required forms.

The manual does not represent all filing requirements for any given application and does not supersede the Building Code, Zoning Resolution, or other applicable rules, regulations, or policies.

## Support Information

For issues or questions on DOB NOW: Build, the manual directs users to submit a help request through `www.nyc.gov/dobnowhelp`.

## Access DOB NOW: Build

The manual recommends Internet Explorer 9.0 and above and directs users to the DOB NOW: Build login page at `www.nyc.gov/dobnow`.

The access page instructs users to:

1. Enter the email ID and password created for the eFiling system.
2. Use the account-access link if they cannot access their account.
3. Use the DOB NOW user-assistance contact link for additional help.

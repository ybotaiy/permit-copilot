# permit-copilot

AI copilot for building permit applications.

## Setup

```bash
uv sync
```

## Usage

```bash
# Show help
python -m src.main --help

# Or via package entry
python -m src --help

# Run the hello command
python -m src.main hello
```

## Development

```bash
# Lint
uv run ruff check .

# Run tests
uv run pytest
```
